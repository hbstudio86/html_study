1. HTML의 역사

1991년, CERN의 팀 버너리스가 개발
1993년, 미국 마크 앤드리슨과 에릭 비나의 Mosaic이 보편화
1995년, IETF에 의해 HTML2.0이 개발
1994년, W3C 결성
1998년, HTML4.0 발표
    CSS와 Javascript가 포함
1999년, HTML4.01 표준
2000년 이후, XHTML 등장
    HTML과 XML을 합친것으로 HTML을 완전히 대체하지는 못함.
2004년, Web Applications 1.0 발표 (WHATWG)
2008년, HTML5 초안 공개
2014년, HTML5가 차세대 웹 표준으로 채택
2016년, 2017년에 각각 HTML5.1과 HTML5.2가 최신표준으로 지정
2019년, HTML Living Standard 표준
    W3C와 WHATWG의 공동 표준 작업

2. 웹 이란 무엇인가?

웹 이란,
세계적 규모의 거미집 형태의 망이라는 뜻으로, 하이퍼링크 개념과 그래픽 환경의 인터페이스를 사용하여 다양한 형태의 데이터와 정보에 접근할 수 있도록 만든 인터넷 서비스를 말한다.

인터넷이란,
1969년 미국 국방성 산하의 고등연구국에서 유사시에도 안정된 정보와 자원 공유를 위해 만든 ARPANET이 그 시초로 오늘날 전 세계적으로 컴퓨터를 연결시켜주는 네트워크 시스템을 말한다.

웹은 1989년에 CERN에서 연구자들이 상호 연구의 신속한 정보 교환과 관리를 위한 개념에서 시작하여 팀 버너스리에 의해 최초로 설계 되었다.
웹 페이지는 HTML과 CSS 그리고 JavaScript를 통해 만들 수 있다.

3. HTML 주요 요소

<br> : 줄바꿈 요소
HTML에 작성한 문서의 개행을 표시하기 위한 요소 입니다.
종료 태그가 필요하지 않습니다.
<hn> : 제목 지정 요소
문서의 제목을 작성할 때 사용합니다. n의 값을 입력하면 글자의 크기가 변경됩니다.
n의 값이 작으면 반대로 글자의 크기가 커집니다.
<img> : 이미지 삽입 요소
문서에 이미지를 입력하기 위해 사용하는 요소입니다.
<a> : 하이퍼링크 요소
문서에서 특정 웹페이지와 연결시켜주는 요소 입니다.
이는 특정 웹사이트, 문서내의 다른 영역, 파일등 여러 형태를 지정할 수 있습니다.
<iframe> : 웹페이지 속에 다른 웹페이지 출력하는 요소
웹페이지속에서 다른 웹페이지를 표시하는데 사용 됩니다.
