        브라우저가 html 문서에 접근할 수 있도록 정의해 놓은 표눚 모델
        html 문서를 분석하고 표시하는 방식
        자바스크립트를 이용하면 프로글매 실행중에 
        웹 문서의 내용 구조 및 스타일에 대한 작업 가능ㅇ
    
        id 속성을 이용한 방법
        getElementByID(아이디명)

        요소명을 이용하는 방법
        getElementByTagName(요소명)
        지정한 요소명을 사용하는 모든 요소를 찾아 리스트 형태로 반환(전체 요소를 찾으니깐)

        클래스명을 이용한 방법
        getElementByClassName(클래스명)
        class = 클래스명 속성을 갖는 모든 요소를 찾아 리스트 형태로 반환
