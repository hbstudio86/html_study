HTML의 역사

  • 1991년, CERN의 팀 버너리스가 개발
  • 1993년, 미국 마크 앤드리슨과 에릭 비나의 Mosaic이 보편화
  • 1995년, IETF에 의해 HTML2.0이 개발
  • 1994년, W3C 결성
  • 1998년, HTML4.0 발표
    CSS와 Javascript가 포함
  • 1999년, HTML4.01 표준
  • 2000년 이후, XHTML 등장
    HTML과 XML을 합친것으로 HTML을 완전히 대체하지는 못함.
  • 2004년, Web Applications 1.0 발표 (WHATWG)
  • 2008년, HTML5 초안 공개
  • 2014년, HTML5가 차세대 웹 표준으로 채택
    2016년, 2017년에 각각 HTML5.1과 HTML5.2가 최신표준으로 지정
  • 2019년, HTML Living Standard 표준
    W3C와 WHATWG의 공동 표준 작업
