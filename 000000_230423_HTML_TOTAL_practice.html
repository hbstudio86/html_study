Web 개요

  1. Internet : 전 세계적으로 연결되어 있는 컴퓨터 네트워크 시스템
  2. 1969년 미국 국방성 산하의 고등연구국에서 ARPANET이 그 시초
  3. World Wide Web : 세계적 규모의 거미집 형태의 망이라는 뜻
    Hyper-link, GUI를 사용한 다양한 형태의 데이터와 정보에 접근할 수 있도록 마든 인터넷 서비스
  4. WEB : 1989년 CERN의 Tim Berners-Lee가 처음 만듦
    HTTP,HTML의 설계도 최초로 만듦
  5. Web : HTML작성된 문서 -> HTTP Protocol -> Web Browser
  6. Web Site : Web page의 "의미있는"묶음

HTML5: 웹 문서 만들기

  1. 줄 바꾸는 요소 : <br>
    forced line break, 텍스트 편집기의 엔터키와 같은 효과 <br/>를 사용해도 됨.
  2. 단락을 나누는 요소 : <p>
    paragraph, 문단을 나누어 독리된 단락을 형성. 여러번 사용해도 한 번만 사용한 효과.
  3. 수평선을 그리는 요소 : <hr>
    horizontal rule, 텍스트 사이에 수평선을 그림. 자동 줄바꿈 기능이 있음
  4. 제목을 지정하는 요소 : <hn>
    heading, 단락의 제목을 지정할 때 사용. n은 1에서 6까지의 숫자로 지정 함
  5. 텍스트를 자유롭게 표현하는 요소 : <pre>
    preformatted text, 편집기에서 작성된 코드의 입력 형태를 그대로 유지함.
